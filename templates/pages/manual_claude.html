1. Brand Foundation

Essência da Marca

"Progresso registrado. Resultados verificados."

Proposta de Valor

MIKE transforma treinos em dados acionáveis. Registre cada série com precisão, visualize seu progresso objetivamente e tome decisões baseadas em evidências.

Personalidade da Marca

PRECISO

Cada número importa. PRs são celebrados com dados, não sentimentos.

DIRETO

Zero floreios. Sem gamification infantil. A recompensa é o progresso real.

CONFIÁVEL

Consistência visual e funcional. O histórico é imutável.

Tom de Voz — Exemplos

Contexto ✅ Usar ❌ Evitar
Sucesso "Treino registrado. +2kg no supino — novo PR." "Arrasou, campeão! 🔥💪"
Erro "Série incompleta. Adicione reps ou carga." "Ops! Algo deu errado :("
Empty State "Nenhum treino registrado ainda. Comece agora." "Cadê os gainz? Vamos começar!"

8. Regras Do/Don't

✓ Usar MIKE Accent para PRs

Reserve a cor laranja para destaques realmente importantes: novos PRs, conquistas significativas.

✗ Usar vermelho como positivo

Vermelho é universalmente associado a erro/perigo. Nunca use para celebrar progresso.

✓ Manter 80% neutros

Maioria da interface deve usar Surface, Border e Text. Cores saturadas são acentos.

✗ Texto pequeno em Accent

Laranja não passa em contraste WCAG para texto <18px. Use apenas em badges bold.

✓ Feedback imediato

Check verde + animação 200ms ao completar ação. Reforço comportamental instantâneo.

✗ Múltiplas cores saturadas

Evite misturar 3+ cores vibrantes na mesma tela. Causa poluição visual.
